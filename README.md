
## Running instructions

1. docker-compose up
2. open browser type following url `http://0.0.0.0:4200/`
3. enter following credentials 
* login: spa_user
* password: spa_pass
4. to use SWAGGER UI enter `http://localhost:8888/swagger-ui.html` and use following credentials
* login: spa_user
* password: spa_pass
